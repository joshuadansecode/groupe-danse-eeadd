📱 Installer l'application

Installez l'application sur votre téléphone pour y accéder comme une vraie app, sans navigateur.

📱 Sur iPhone (Safari)
1. Ouvrez le fichier dans Safari
2. Appuyez sur le bouton Partager ↑
3. Choisissez "Sur l'écran d'accueil"
4. Confirmez avec "Ajouter"
🤖 Sur Android (Chrome)
1. Ouvrez le fichier dans Chrome
2. Appuyez sur les 3 points ⋮
3. Choisissez "Ajouter à l'écran d'accueil"
4. Confirmez
